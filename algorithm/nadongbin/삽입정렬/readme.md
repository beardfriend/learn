삽입 정렬은 위 문제를 풀 때 각 숫자를 **적절한 위치에 삽입**하는 방법으로 문제를 해결한다.

삽입과 버블은 이미 정렬이 이루어진 상태임에도 불구하고 반복과 정렬을 진행한다는 특징이 있었다.
삽입정렬은 **필요할 때만 위치를 바꾼다.**

O(N^2)의 알고리즘을 가진 정렬 중에서는 그나마 효율적인 방법을 가져간다.
